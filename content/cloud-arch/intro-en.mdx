# Cloud Architecture Overview

Cloud Architecture is the design of components and processes required to build applications on the cloud.

## Key Components:
1. **Compute**: EC2, Lambda, etc.
2. **Storage**: S3, EBS.
3. **Networking**: VPC, Route53.
4. **Security**: IAM, KMS.

A good system design must ensure High Availability and Scalability.
