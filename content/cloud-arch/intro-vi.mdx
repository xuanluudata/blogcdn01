# Tổng quan về Cloud Architecture

Cloud Architecture là việc thiết kế các thành phần và quy trình cần thiết để xây dựng các ứng dụng trên đám mây.

## Các thành phần chính:
1. **Compute**: EC2, Lambda, v.v.
2. **Storage**: S3, EBS.
3. **Networking**: VPC, Route53.
4. **Security**: IAM, KMS.

Thiết kế một hệ thống tốt cần đảm bảo tính sẵn sàng cao (High Availability) và khả năng mở rộng (Scalability).
