**The Nice Dataへようこそ**。私はこのブログの著者である **Nguyen Xuan Luu** です。

## **私の使命**

私は、データはあらゆる現代ビジネスにおいて最も価値のある資産であると信じています。私の目標は、組織が堅牢でスケーラブルかつ高度にセキュアなデータ基盤を構築できるようなソリューションに貢献し、AIとアナリティクスを通じて潜在的な価値を引き出すことです。

## **専門分野**

- **システムアーキテクチャ (System Architecture / System Architect)：** オンプレミスおよびクラウドプラットフォーム上での分散型・高負荷システムの設計。
- **データベース管理 (Database Administration / DBA)：** RDBMSおよびNoSQLシステムのパフォーマンス最適化とデータセキュリティの確保。
- **データエンジニアリング (Data Engineering)：** シンプルで効率的なデータパイプライン（ETL/ELT）の構築。
- **データガバナンス (Data Governance)：** データ品質を確保するためのプロセスや標準の簡素化と実効性の向上。
- **人工知能 (Artificial Intelligence / AI Engineering)：** 実世界のビジネス課題へのAIモデルの適用。

## **なぜ「The Nice Data」なのか？**

この名前は私の仕事に対する哲学を表しています。データは単に「ビッグ」であるだけでなく、「ナイス」であるべきです。「ナイス」なデータセットとは、クリーンで、構造化されており、理解しやすく、価値を生み出す準備が整っているものです。

このブログで共有する内容が、データの世界を習得するあなたの旅に役立つことを願っています！

## **自己紹介**

私はベトナム・ハノイで働くエンジニアです。既婚で、幸せな小さな家族がいます。

2015年、私はハノイ工科大学で電子通信工学を学びました。ここで科学と技術への情熱を追求しました。専門知識やスキル、そしてテクノロジー分野におけるエンジニアとしての思考を身につけました。ハノイ工科大学は、多くの素晴らしい思い出や友情、そして貴重な人間関係が生まれた場所でもあります。

2020年、私は優秀な成績で工学学位を取得し、VNPT VASで働き始めました。この組織は、VNPTグループの通信プラットフォームおよびデジタルサービス上で付加価値サービスの研究・開発・運用を行っています。私はシステム監視担当として24時間365日のシフト勤務からスタートしました。その後、システムエンジニアとして通信ユーティリティ分野のプラットフォームサービスの運用を担当しました。同時に、ハノイ工科大学の修士課程に進学し、大学からの全額奨学金を受けて学びました。

2022年、私は通信工学の修士号を良好な成績で修了しました。研究志向から応用工学へと完全にシフトし、学術研究の道からは離れました。その後もVNPT VASで勤務を続けながら、データ関連の業務にも携わり、ビジネスレポーティングのためのデータプラットフォーム構築に参加しました。

2024年、私は複雑な分野であるOracleデータベース管理に特化しました。トレーニングを修了し、OCP認定試験に合格し、世界でも最先端の技術を用いたデータベースシステムに携わりました。同時に、企業内でのビッグデータソリューションの研究と実装において主導的な役割を担いました。

2026年、私とチームは企業向けビッグデータプラットフォームの開発を完了しました。このプラットフォームは仮想化環境上に構築され、Apache Airflow、Apache NiFi、Apache Spark、MinIO、SingleStore、Metabaseといった主要技術を含むオープンソースおよび無償ソリューションを活用しています。

2024年から2026年にかけて、私はスキルを向上させ、新たな要件に対応し将来の課題に備えるため、クラウドおよびデータ分野の認定試験に合格しました。取得した認定は以下の通りです：

- Oracle Database Administration 2019 Certified Professional (OCP)
- Oracle Certified Professional: MySQL 8.0 Database Administrator
- Google Cloud Professional Cloud Architect
- Google Cloud Professional Data Engineer
- Microsoft Azure Solutions Architect Expert (AZ-305)
- Microsoft Azure Administrator Associate (AZ-104)
- Microsoft Azure Database Administrator Associate (DP-300)
- Microsoft Fabric Data Engineer Associate (DP-700)
- Microsoft Fabric Analytics Engineer Associate (DP-600)
- Snowflake Platform Associate

これらの認定取得後、これまでの知識・経験・資料を体系化し、ブログ **The Nice Data** を構築しました。**The Nice Data** があなたに価値ある洞察をもたらすことを願っています。
