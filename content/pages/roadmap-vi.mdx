# Roadmap

Lộ trình phát triển và học tập trong lĩnh vực Kiến trúc Dữ liệu.

## 1. Nền tảng (Foundations)
- SQL & Relational Databases
- Programming (Python/Java/Go)
- Linux & Shell Scripting

## 2. Kỹ thuật Dữ liệu (Data Engineering)
- ETL/ELT Pipelines
- Data Warehousing (BigQuery, Snowflake)
- Batch & Stream Processing (Spark, Flink)

## 3. Kiến trúc Dữ liệu (Data Architecture)
- Data Modeling (Kimball, Data Vault)
- Data Lakehouse Architecture
- Modern Data Stack
