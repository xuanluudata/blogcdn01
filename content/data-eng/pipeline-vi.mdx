# Xây dựng Data Pipeline với Python

Data Pipeline là xương sống của bất kỳ hệ thống dữ liệu nào.

## Các bước thực hiện:
1. **Extract**: Lấy dữ liệu từ nguồn (API, DB, Files).
2. **Transform**: Làm sạch và biến đổi dữ liệu.
3. **Load**: Đưa dữ liệu vào Data Warehouse.

Sử dụng các công cụ như Airflow hoặc Prefect để quản lý workflow.
