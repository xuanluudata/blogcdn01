# Golden Rules for DBA

Database Administration (DBA) requires meticulousness and adherence to strict processes.

## 1. Regular Backups
Always ensure you have the latest backup and have tested its restorability.

## 2. Query Optimization
Use indexes wisely and analyze Execution Plans.

## 3. Data Security
Grant permissions based on the Least Privilege principle.
